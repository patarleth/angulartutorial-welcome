yo angular -

Open the secure shell and run -

    cd ~
    mkdir first-app
    cd first-app
    yo angular first

    The angular-generator asks 3 questions

    1. do you want to use Sass and Compass (press n for no)
    2. do you want to use Twitter Bootstrap (press y for yes)
    3. which angular features you wish to use (press enter)

Open the secure shell to you home directory (~) and create a new project folder named first-app -
    cd ~
    mkdir first-app

FYI - feel free to name the folder anything you like, I'm going with first-app (no spaces in the name please).

If this is the first time using yo angular I suggest keeping it simple, leave the name alone.

Enter the directory in the shell
    cd first-app
If you scaffold the app (using yeoman yo!) like so
    yo angular first-app
Yeoman (yo!) tries to be cute and adds 'App' to the end of the name used in the javascript. Soooo the name becomes -
    firstAppApp
So if having your app named AppApp bugs you, run the generator without App -
    yo angular first
firstApp
The angular generator will ask 3 questions
  1. do you want to use Sass and Compass (press n for no)
  2. do you want to use Twitter Bootstrap (press y for yes)
  3. which angular features you wish to use (press enter)
Why answer no, yes and enter?
  1. Sass and Compass - read about it here http://compass-style.org/ sass is ... interesting, but not important for the purpose of this tutorial.
  2. Bootstrap has a bunch of great css styles and makes bad looking apps look, well better...
  3. We're learning angular, so why not install all the toys ;)
