Online IDE shiftedit.net

A simple and free IDE with an integrated sftp and secure shell. A great tool, esp. for price. Plenty of other online options available - cloud9, codebox, neutron drive, codeenvy, etc.

Open chrome and navigate to

    https://shiftedit.net/home#

Setup a shiftedit to point to my test server on azure

    1. first send me an email (pat.arleth) for a login (yo_1 is used below)
    2. open https://shiftedit.net/home# and click create account. I used the google signin integration.
    3. set name and domain to angulartutorial.cloudapp.net
      the dialog that pops up on initial login should look like this:


      click Test Connection, and Save

      if you accidently closed the "Site Settings" window, no worries - just click the three horizontal line icon top left and select "New Site" or "Edit Site".


    4. configure ssh by clicking the "Secure Shell" link at the bottom of the page and the the PLUS icon.
