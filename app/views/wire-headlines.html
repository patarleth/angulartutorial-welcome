wire up the espn.js factory to the headlines.js controller

in app/scripts/controllers/headlines.js modify the constructor to include the espn factory name like so 

'use strict';
angular.module('tutorialApp').controller('HeadlinesCtrl', function ($scope, espn) {
  espn.espnnow().then(function (result) {
    $scope.espnnow = result;
  }, function (err) {
    console.log(err);
  });
});

now, whenever the HeadlinesCtrl is created, it will call espn.espnnow() asychonously. When it is successful, sets a variable available in

    app/views/headlines.html
or log an error if the REST call throws an erorr or returns an http status code indicating failure.
