declaring a REST $resource in an angularjs factory

This will have to go mostly unexplained for now.

For more information - checkout my sample git project on code.espn.com/patarleth/angular-tutorial

Short version, if you genereate a factory using yo
  • yo angular:factory espn
edit the factory js file
  • app/scripts/services/espn.js
and add
  var espnNowResource = $resource(
    'http://api.espn.com/v1/now?apikey=v37fddaw8ge4knf8yqq7nnd9', 
    { callback: 'JSON_CALLBACK' }, { get: { method: 'JSONP' } 
  });
  var espnnow = function () {
  	return espnNowResource.get({}).$promise;
    };
    
    ...(dont' actually paste ... btw)
    
    // Public API here 
    return {                                                                                            
    espnnow: function () {                                                                            
      return espnnow();                                                                               
    }
  };
The factory will now have a single REST resource definition used anytime espn.espnnow() is called.
