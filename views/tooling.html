recommended tools

Besides Crockford, the only tool you'll need to dev angular apps is your favorite text editor.

Just download the .js dependencies from https://angularjs.org/ and go.

It's dangerous to go alone - I suggest you take these
  • Webstorm 8 - Great (NOT FREE) IDE with command line tool support
  • grunt - command-line task runner to run unit tests, minify javascript for deployment, or anything else you want to automate as you develop your application
  • bower - command-line javascript dependency management tool. downloads and versions js libs, freeing you from downloading them by hand
  • yeoman - command-line code scaffolding tool. much more about this in a minute

The three command-line tools - yo, grunt and bower were written on unix boxes for unix boxes, windows is a second class citizen at best. Ports exist but I highly recommend running yo on a unix VM.
